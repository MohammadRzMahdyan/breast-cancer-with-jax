دیتاست سرطان 
دیدن جریان اطلاعات قسمت مفیدی هست که میشه تو این کد ها دید.
قسمت هایی که جریان اطلاعات رو نشون میدادن بدون فریم ورک نوشته شدن.
دیتاست نیازی به شبکه عصبی عمیق یا حتی با یک لایه ی مخفی نداشت , پرسپترون دقت خوبی میداد.
برای این دیتاست از `sigmoid_binary_cross_entropy` استفاده شده که مشکل انفجار گرادیان رو در این دیتاست رفع میکنه.
این کد با `JAX` نوشته شده تا حدی کنترل پذیری بالایی داره .
کاملا `pure function`!
